# Analista-para-el-operador-de-telecomunicaciones-Megaline-Caso-tarifas-de-prepago-Surf-y-Ultimate-
Realizar un análisis preliminar de las tarifas de prepago (Surf y Ultimate) de la empresa de telecomunicaciones Megaline para determinar cuál de los dos planes genera más ingresos, con el fin de ayudar al departamento comercial a ajustar el presupuesto de publicidad.
